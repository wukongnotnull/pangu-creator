| 类型 | 模板文件 | 适用场景 |

## 模板选择指南

```
用户输入是什么类型？
├── 人物（芒格、巴菲特、马斯克）
│   ├── 蒸馏 → 使用 person-skill-template.md
│   └── 超越 → 使用 person-plus-template.md
├── 内容（《反脆弱》、《原则》、TED课程）
│   ├── 蒸馏 → 使用 content-skill-template.md
│   └── 超越 → 使用 content-plus-template.md
├── 思想（长期主义、第一性原理、杠铃策略）
│   ├── 蒸馏 → 使用 idea-skill-template.md
│   └── 超越 → 使用 idea-plus-template.md
└── 现象（瑞幸咖啡增长、内卷、躺平）
    ├── 蒸馏 → 使用 phenomenon-skill-template.md
    └── 超越 → 使用 phenomenon-plus-template.md
```

## 使用方法

1. 根据蒸馏/超越需求选择对应模板
2. 在 `references/templates/` 目录下查看模板详情
3. 复制模板内容到目标 Skill 目录的 `SKILL.md`
4. 根据模板结构填充内容
